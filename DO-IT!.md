## 02-2. 문자형 자료형

### 문자열 포매팅

* 딕셔너리

```python
>>> d = {'name' : '조흔지', 'age' : 30}
>>> f"나는 {d['name']}이고, 나이는 {d['age']}입니다."
'나는 조흔지이고, 나이는 30입니다.'
```

* 정렬

```python
>>> f'{"왼쪽":<10}'
'왼쪽        '

>>> f'{"가운데":^10}'
'   가운데    '

>>> f'{"오른쪽":>10}'
'       오른쪽'

>>> f'{"채우기":*^10}'
'***채우기****'
```

* 소수점

```python
>>> y = 0.123456789
>>> f'{y:0.4f}'
'0.1235'
```

* {}표시

```python
>>> f'{{ "괄호표시" }}'
'{ "괄호표시" }'
```

* 실전 연습  : `!!!python!!!` 만들기

```python
>>> f'{"python":!^12}'
'!!!python!!!'

>>> "{0:!^12}" .format("python")
'!!!python!!!'
```
